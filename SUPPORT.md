# 支持

请使用与请求相匹配的最轻量的支持路径：

- **使用问题：** 从 [USAGE.md](./USAGE.md) 和 [rust/README.md](./rust/README.md) 开始。
- **Bug 或回归：** 创建 GitHub issue，附上命令、操作系统/shell、预期行为、实际行为以及相关的非机密输出。
- **安全问题：** 遵循 [SECURITY.md](./SECURITY.md)，不要创建公开 issue。
- **社区讨论：** 使用 [README.md](./README.md) 中链接的 UltraWorkers Discord。

寻求帮助时，请包含：

```text
claw --version
claw doctor
操作系统和 shell
你执行的命令
```

在分享输出之前，请编辑掉 API 密钥、bearer token、私有提示词、会话记录以及可能泄露敏感信息的本地路径。
